## 简介

CSS3即层叠样式表(Cascading StyleSheet)的最新标准，在原有基础上新增了选择器、框模型、背景和边框、文本效果、2D/3D 转换、动画、多列布局、用户界面

## CSS3选择器

作为初入CSS3系列的第一部分，将会首先为大家讲解CSS中的选择器,在CSS2.0标准下的时候，CSS的选择器样式太过于单一，无法直接精准的为元素设置样式，在进行复杂项目的开发的时候常常要用大量的class，id标签，CSS3.0版本引入了新的选择器使得开发效率大大提高。

### 属性选择器

html格式如下

	<div>div1</div>
	<div age="11">div2</div>
	<div age="12 13">div3</div>
	<div age="14 15 12">div4</div>
	<div age="16 17">div5</div>
	<div age="18-19">div6</div>

通用css样式如下

	div{width:80px;height:80px;border:1px solid #000;float:left;}

E[attr] 筛选出E标签attr属性的元素,例子如下:

	div[age]{
		bakcground:#c0c0c0;
	}

结果如下：

### 伪类与伪元素

来看下W3C对伪类和伪元素的定义:


	CSS伪类：用于向某些选择器添加特殊的效果。

	CSS伪元素：用于将特殊的效果添加到某些选择器。

#### 伪类
如E:hover伪类就是为E选择器被鼠标覆盖添加新的效果,同样CSS3新增了几项新的伪类,下面将为大家介绍。

E:target 为E中target指当锚链接URL所指向的元素添加样式

html

	<div id="box1"></div>
	<div id="box2"></div>

css 
	
	div:target{
		background:#c0c0c0;
	}

在浏览器的地址栏末尾添加上"#box1",就能得到如下结果

<div >
<div class="box active"></div>
<div class="box"></div>
<div style="clear:both;"></div>
</div>


下面几例与hover属性类似，就不一一举例:

E:disabled 表示不可点击的表单控件

E:enabled 表示可点击的表单控件

E:checked 表示已选中的checkbox或radio



#### 伪元素:


E:first-line 表示E元素中的第一行

E:first-letter 表示E元素中的第一个字符

例子如下:

html

	<p>
		JavaScript一种直译式脚本语言，是一种动态类型、弱类型、基于原型的语言，内置支持类型。它的解释器被称为JavaScript引擎，为浏览器的一部分，广泛用于客户端的脚本语言，最早是在HTML（标准通用标记语言下的一个应用）网页上使用，用来给HTML网页增加动态功能
	</p>

css

	p:first-line{
		background:#c0c0c0;
	}
	p:first-letter{
		font-size:25px;
		color:rgb(128,128,192);
	}

<p class="text1" style="border:1px solid #000;">
	JavaScript一种直译式脚本语言，是一种动态类型、弱类型、基于原型的语言，内置支持类型。它的解释器被称为JavaScript引擎，为浏览器的一部分，广泛用于客户端的脚本语言，最早是在HTML（标准通用标记语言下的一个应用）网页上使用，用来给HTML网页增加动态功能
</p>


E::before 生成内容在E元素前

E::after 生成内容在E元素之后

这两个伪元素经常与content属性一起使用,下面为使用的例子:

css

	p::before{
		content:"AllenYang";
		color:rgb(128,128,192)
	}
	P::after{
		color:rgb(128,128,192)
		content:"AllenYang";
	}

<p class="text2" style="border:1px solid #000;">
	JavaScript一种直译式脚本语言，是一种动态类型、弱类型、基于原型的语言，内置支持类型。它的解释器被称为JavaScript引擎，为浏览器的一部分，广泛用于客户端的脚本语言，最早是在HTML（标准通用标记语言下的一个应用）网页上使用，用来给HTML网页增加动态功能
</p>

### 其他选择器

E:not(s) 表示E元素不被匹配,例子如下

html

	<div>div1</div>
	<p>p1</p>
	<div id="#box">div2</div>
	<div>div3</div>
	<p>p2</p>
	<p>p3</p>
	<div>div4</div>
	
css

	p{
		border:1px solid #000;
		width:80px;
		height:80px;
		float:left; 
		margin:0 20px 20px 0;
	}
	div:not(#box){
		background:#c0c0c0;
	}
